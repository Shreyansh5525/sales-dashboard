# 📊 Business Sales Insights Dashboard

An interactive data analysis dashboard built using **Python, Pandas, and Streamlit** to analyze sales data and generate business insights.

---

## 🚀 Features
- 📌 Interactive dashboard with filters (Region-based)
- 📊 Key metrics (Total Sales, Total Profit)
- 📦 Sales analysis by category
- 📈 Monthly sales trend visualization
- 🏆 Top 5 products analysis

---

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Seaborn
- Matplotlib

---

## 📂 Dataset
- Superstore Sales Dataset (Kaggle)

---

## ▶️ How to Run Locally


pip install -r requirements.txt
streamlit run app.py
